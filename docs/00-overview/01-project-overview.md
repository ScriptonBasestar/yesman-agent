# Project Overview

Yesman-Claude는 Claude Code Headless 모드를 활용한 API 중심의 자동화 및 모니터링 플랫폼입니다. Agent와 Dashboard간 실시간 통신을 통해 확장 가능한 아키텍처를 제공합니다.

## 🎯 프로젝트 개요

### 목적

Claude Code 자동화를 통해 개발자의 생산성을 향상시키고, 반복적인 작업을 자동화하여 더 중요한 작업에 집중할 수 있도록 돕습니다.

### 핵심 가치

- **자동화**: Claude Code 상호작용의 완전 자동화
- **모니터링**: 실시간 세션 및 프로젝트 상태 모니터링
- **효율성**: 스마트 캐싱과 성능 최적화
- **확장성**: 플러그인 시스템과 API 기반 확장
- **사용성**: 직관적인 CLI 및 GUI 인터페이스

## 🚀 주요 기능

### 1. Claude Code Headless 자동화

- **Headless SDK**: Claude CLI 기반 완전 자동화 (Interactive 모드 백업 지원)
- **API 통합**: 8개 REST 엔드포인트로 Agent 생명주기 관리
- **실시간 스트리밍**: JSON 기반 작업 진행 상황 모니터링
- **보안 샌드박스**: 워크스페이스 격리 및 명령 검증

### 2. Agent 및 세션 관리

- **Agent 생명주기**: 생성, 모니터링, 종료 자동화
- **tmux 통합** (옵셔널): 터미널 기반 세션 지원
- **YAML 설정**: 유연한 템플릿 시스템
- **실시간 모니터링**: WebSocket/SSE 기반 상태 업데이트

### 3. 프로젝트 모니터링

- 8개 카테고리 건강도 분석
- 실시간 대시보드
- 성능 지표 추적
- 활동 히트맵

### 4. 멀티 인터페이스

- **CLI Interface**: 강력한 명령줄 인터페이스로 모든 기능 접근
- **SvelteKit Web Dashboard**: 브라우저 기반 모니터링 및 제어
- **Tauri Desktop Application**: 네이티브 데스크톱 앱으로 시스템 통합
- **REST API (FastAPI)**: 고성능 비동기 백엔드 서비스

### 5. AI Provider Management

- **자동 검색**: 시스템에 설치된 AI 도구 자동 탐지 (Claude Code, Ollama, etc.)
- **다중 검색 방식**: CLI 도구, 환경변수, 실행 중인 서비스 검색
- **상태 관리**: not_installed, detected, registered, active 상태 추적
- **크로스 플랫폼**: Tauri 백엔드를 통한 시스템 레벨 접근

## 🏗️ 아키텍처 개요

### 컴포넌트 구조

```
┌──────────────────┐    ┌──────────────────┐    ┌──────────────────┐
│  SvelteKit Web   │    │   FastAPI REST   │    │  Tauri Desktop   │
│   Dashboard      │    │      API         │    │      App         │
└──────────────────┘    └──────────────────┘    └──────────────────┘
         │                       │                       │
         └───────────────────────┼───────────────────────┘
                                 │ (WebSocket/SSE)
         ┌───────────────────────────────────────────────┐
         │              Agent Engine                     │
         │  ┌─────────────┐  ┌─────────────┐  ┌─────────┐ │
         │  │ HeadlessAdp │  │ ClaudeAgent │  │ Workflow│ │
         │  └─────────────┘  └─────────────┘  └─────────┘ │
         └───────────────────────────────────────────────┘
                                 │
         ┌───────────────────────────────────────────────┐
         │              System Layer                     │
         │  ┌─────────────┐  ┌─────────────┐  ┌─────────┐ │
         │  │Claude CLI   │  │tmux(옵셔널) │  │ Storage │ │
         │  └─────────────┘  └─────────────┘  └─────────┘ │
         └───────────────────────────────────────────────┘
```

### 데이터 흐름 (API 중심)

1. **Agent 생성**: REST API → Claude CLI Headless 모드 초기화
2. **작업 실행**: JSON 스트리밍 → 실시간 진행 상황 추적
3. **상태 동기화**: WebSocket/SSE → Dashboard 실시간 업데이트
4. **결과 수집**: Claude CLI 출력 → 구조화된 응답 반환
5. **리소스 정리**: Agent 종료 → 자동 리소스 해제

## 💻 기술 스택

### 백엔드

- **Python 3.11+**: 핵심 로직 및 Agent 엔진
- **FastAPI**: 비동기 REST API 서버
- **Claude CLI**: Headless 모드 SDK 통합
- **uv**: 패키지 관리 및 실행 환경
- **tmuxp/libtmux** (옵셔널): 터미널 세션 지원

### 프론트엔드

- **Tauri**: 크로스 플랫폼 데스크톱 프레임워크
- **SvelteKit**: 반응형 UI 프레임워크
- **TypeScript**: 타입 안전성
- **Tailwind CSS + DaisyUI**: 스타일링

### 인프라 및 도구

- **uv**: 통합 Python 패키지 관리 (pip 대체)
- **pnpm**: Node.js 패키지 관리
- **Ruff**: 린팅 및 포매팅 도구
- **WebSocket/SSE**: 실시간 통신
- **Docker** (옵셔널): 컨테이너 배포

## 🚦 시작하기

### 빠른 시작

```bash
# uv를 사용한 설치
uv sync

# API 서버 시작
uv run python -m uvicorn api.main:app --host 0.0.0.0 --port 10501

# 대시보드 실행
make dashboard  # 또는
uv run make dashboard-web
```
